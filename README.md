## 📌 About The Project

**SkillMatrix** is a Java Full Stack web-based Online Examination System designed to conduct secure and structured assessments for students.  
The platform provides dedicated admin and student modules to manage exams, questions, results, and performance tracking efficiently.

Developed using **Java Servlets, JSP, JDBC, MySQL, HTML, CSS, and Apache Tomcat**, the application follows an MVC architecture to ensure modularity, scalability, and maintainability.

---

## 🎯 Core Functionalities

### 👨‍💼 Admin Module
- ➕ Create, update, and delete exams
- 📝 Add multiple-choice questions
- 👥 Manage student accounts
- 📊 View results and performance analytics

### 👨‍🎓 Student Module
- 🔐 Secure login authentication
- 🧠 Attempt timed online exams
- 📈 View scores and result history
- 📄 Instant result generation
